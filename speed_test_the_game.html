Background: Speed Test the Game is based on a popular Finnish reaction game from late 80's / early 90's. It became famous from the TV game show Speden Spelit.

Instructions: Push-buttons light up at an increasing speed. Press them following the correct sequence. When you make a mistake the game ends.

Controls: mouse/keyboard (configurable for keys A-Z and 0-9)
